## 🚀 Apa itu Premium Remote VPN?
Skrip ini adalah sebuah *auto-installer* multi-fungsi yang dibuat khusus dengan antarmuka CLI yang canggih (berwarna dan informatif). Skrip ini otomatis membaca sistem operasi, IP, dan RAM VPS Anda sebelum menawarkan instalasi berbagai protokol VPN populer.

## ✨ Fitur Tersedia
- 🟢 Instalasi **XRAY / V2RAY** (Mendukung VMess, VLess, Trojan).
- 🟢 Instalasi **SSH & WebSocket** (Dropbear, Stunnel4, SSLH).
- 🟢 Instalasi **OpenVPN & WireGuard** (Standar VPN yang aman).
- 🟢 Optimasi jaringan dengan **TCP BBR** (Meningkatkan kecepatan internet VPS).

## 🛠️ Cara Menginstal di VPS Anda
Pastikan Anda sudah login ke terminal VPS Anda sebagai **root**, lalu jalankan perintah ini:

```bash
wget https://raw.githubusercontent.com/edholabs/RemoteVpn-AutoInstaller/main/install.sh
sudo bash install.sh
```
